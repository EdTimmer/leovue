Leoviewer is written in Vue.js, and Vue components can be added to content panes.

At the moment only the Vue.js Youtube component is supported by default. However, you can easily add any other Vue.js component by declaring it in the main.js file like so:

import VueYouTubeEmbed from 'vue-youtube-embed'
Vue.use(VueYouTubeEmbed)

In your html content, add the Vue.js Youtube component like so:

<youtube :video-id="'pRnQFGskg9w'" :player-width="320" :player-height="195"></youtube>
