Leo Viewer is a web app that can display files created with Leo, the open source outlining editor and IDE.

Leo has features that make it particularly suitable for documenting code and other complex documents.

Using Leo, you can create an outlined document like the one you are viewing. Content nodes can contain text, html, markdown or code. Content nodes can also point to other documents or websites.

Using Leo Viewer, users can view your document as a normal webpage without having to install Leo.
